Getting started with C++

Lets start with the most basic task in programming: "Hello, world!"

"Hello, world!" is the simplest program to show how the language works. It is usually the first thing new programmers write because it displays how a language is structured.

Also from now on you will be given a snippet of code, your task is to complete the code with the missing line, but before that, lets break down the code:

  • #include <iostream> is a header file library that lets us work with input and output objects, such as cout. Header files add functionality to C++ programs.
  • using namespace std means that we can use names for objects and variables from the standard library.

It's fine you don't understand these two, that is just almost always used for the code to work.

  • int main() is a function, any code inside its curly brackets {} will be executed.
  • cout (pronounced "see-out") is an object used together with the insertion operator << to output text. In this example the output will be "Hello World!"
  • return 0 ends the main function.
