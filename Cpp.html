C++

C++, created by by Bjarne Stroustrup in the 1980s, is a general-purpose programming language, it is an extension of the C programming language and provides additional features for object-oriented programming. C++ is a common programming language that is used in development of operating systems, desktop applications, games, data structures and many other things.

Key features of C++ include:

  • Object-oriented
  • Multi-threading
  • Memory management
  • Case-sensitive
  • Simple

Many belive C++ is a "dying language" because of it's age and tend to underrate how poerful it can be. Even so, C++ is commonly used in computer science education as a first language for students to learn.
